pixelRatioが2のRetinaディスプレイにおける画像の表示

今見ているモニターのpixelRatioは です。

例1

400x200pxの解像度72pxの画像

400x200pxで表示され少しぼやける

上記画像のスクショ

800x400pxのサイズで生成され、だいぶぼやけて表示される

上記スクショに400x200pxのスタイルを当てる

だいぶぼやけが解消される

最初の画像に200x100pxのスタイルを当てる

ぼやけが解消される

例2

400x400pxの解像度300pxの画像

少しぼやけて表示される

上記画像に200x200pxのスタイルを当てる

ぼやけが解消される

Retinaディスプレイでもぼやけさせない方法

画像を倍サイズで作っておき表示時に1/2にする

image要素に対しては上記のような処置をし、背景画像に対してはbackground-sizeを用いる。


      background-size: 200px 100px;

画像を2種類作っておきメディアクエリで出し分ける


      @media screen and (min-resolution: 2dppx) { /* ... */ }
      @media screen and (resolution >= 2dppx) { /* ... */ }
